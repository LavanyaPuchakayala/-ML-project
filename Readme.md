# Cyber Threat Analysis on Android Apps Using Machine Learning
# Skills Used

##  Project Overview
This project focuses on analyzing cyber threats in Android applications using Machine Learning techniques. The system identifies malicious applications and predicts potential security threats based on application behavior and permissions.
##  Objective
To develop an intelligent threat detection system for Android apps using Machine Learning algorithms for improved cybersecurity analysis.
## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
## Algorithms Used
- Random Forest
- Decision Tree
- Logistic Regression
- Ensemble Learning
## Features
- Android app threat analysis
- Malware detection
- Data preprocessing
- Feature extraction
- Threat prediction using ML models
- Security risk classification
## Outcome
Successfully developed a Machine Learning-based system for detecting cyber threats and analyzing malicious Android applications with improved prediction accuracy.
